## Wishpond Test Suite

## Installation

* Clone the repo

```
mkdir ~/wishpond-test-suite
cd ~/wishpond-test-suite
git clone git@github.com:hinawishpond/wishpond-test-suite.git
```

* Bundle install

```
bundle install
```

## Usage

* To run all tests

```
rake
```

* To test a single page (e.g; Login Page/Authentication)

```
rspec authentication_spec.rb
```
